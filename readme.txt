== Description ==

Woocommerce Currency Converter is the WooCommerce Multi Currency plugin that allows your site visitors to switch products prices currencies according to set currencies rates in real-time. Your site visitors can pay in the selected currency. As we all know that exchange rates change constantly so, it is difficult for any e-commerce store or wordpress website to keep updating the pricing plans in different currencies. This WooCommerce plugin provides an easier way to let your customers switch between currencies according to their preferences.

###THE PLUGIN FEATURES: 

###FREE FEATURES

&#9658; **Add Two currencies from the back-end.**
&#9658; **Add exchange fee for the selected currency.**
&#9658; **Users can change the currency on shop, cart, and checkout pages.**
&#9658; **Easy to use for both merchants and customers.**
&#9658; **Admin can manually add currency rates.**
&#9658; **Currencies rated will be updated automatically.**
&#9658; **Enable/disable your selected currency visibility on the frontend.**
&#9658; **Available as a widget and works on any WooCommerce website with a widget area.**
&#9658; **Inline styling option for currency switcher to change its background color and font color.** 

###PRO VERSION
&#9658; **Add unlimited multiple currencies from the back-end.** 
&#9658; **Admin can link payment gateways for selected currencies.**
&#9658; **Allow users to select countries for the selected currency. This currency will be selected automatically for the users who belong to these countries.**
&#9658; **Update currency exchange rates on predefined time intervals via the Open Exchange Rates API.**
&#9658; **Auto Update Interval**:This feature of WooCommerce Multi Currency Switcher allows you to add time period to auto-update rates manually. You can set the interval in seconds i.e. 3600 seconds is equal to one hour.
&#9658; **Send Email Notification**:Multi Currency Switcher allows you to send email notifications to the admin email when currency values are updated. An email notification will be sent to the admin whenever currency rates will be updated automatically.

1. Activate the plugin through the 'Plugins' menu in WordPress
